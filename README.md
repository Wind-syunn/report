## 简介

> 项目基于 vue & vant 搭建单页面实例

> 数据报告 -- 关键词: 新基建

## 目录结构

```
cli
┬
├── public
│  └── index.html     模板页面
├── src							  页面逻辑代码
│  ├── api        		接口配置信息
│  ├── assets        	静态资源
│  ├── components     全局共用组件
│  ├── router         前端路由
│  ├── store          全局状态
│  ├── style          全局样式
│  ├── utils          工具函数等
│  └── views          每个页面(视图)
├── package.json
└── vue.config.js
```
